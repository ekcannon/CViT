basic sequence coordinate (kbp) example
----------------------------------------------

An example of psuedochromosomes displayed with sequence coordinates. Both 
strands are shown.

Displays chromosome ranges, approximate location of several knobs and some 
made-up positions and ranges.


Execute:
  PNG output
  $ perl cvit.pl -c examples/Zm-kb/cvit_kb.ini -o examples/Zm-kb/simple_kb examples/Zm-kb/ZmChrs_bp.gff examples/Zm-kb/features.gff
  SVG output
  $ perl cvit.pl -i svg -c examples/Zm-kb/cvit_kb.ini -o examples/Zm-kb/simple_kb examples/Zm-kb/ZmChrs_bp.gff examples/Zm-kb/features.gff
